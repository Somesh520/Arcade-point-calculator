### Step 2: Deploy on Vercel
1. Go to [Vercel.com](https://vercel.com) and login with GitHub.
2. Click **"Add New"** > **"Project"**.
3. Import `Somesh520/Arcade-point-calculator`.
4. **Project Name**: Use a keyword-rich name like `google-arcade-calculator-2026` (Important for SEO!).
5. Click **Deploy**.

### Step 3: Get Indexed (Google Search Console)
1. Copy your new Vercel URL (e.g., `https://google-arcade-calculator-2026.vercel.app`).
2. Go to **Google Search Console**.
3. Add Property > URL Prefix > Paste your URL.
4. Go to **"URL Inspection"** (Top Search Bar).
5. Paste your URL again and click **"Request Indexing"**.
